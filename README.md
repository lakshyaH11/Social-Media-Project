# SQL-SocialMediaProject
Developed an Meta-style social media analytics project using a relational SQL database. Analyzed users, posts, likes, comments, hashtags, and follows to identify engagement trends, top creators, content performance, and influencer potential. Delivered insights for targeted marketing and user retention.
